# Kiwi Digital — Website

Modern, dark-themed website for Kiwi Digital Ltd built with [Astro](https://astro.build) + Tailwind CSS.

## Stack

- **Astro v5** — Static site generation with island architecture
- **Tailwind CSS v3** — Utility-first styling with custom design system
- **TypeScript** — Type-safe development
- **Zero JavaScript frameworks** — Pure Astro components, minimal client JS

## Features

- Dark theme with animated gradient backgrounds
- Glass-morphism UI components
- Scroll-triggered animations (Intersection Observer)
- Parallax hero section with orbiting elements
- Responsive navigation with mobile drawer
- SEO optimised (meta tags, Open Graph, structured data, sitemap)
- Contact form (Netlify Forms compatible)
- Custom Tailwind colour palette (Kiwi green + Ocean blue)
- Performance-first: minimal JS, static HTML, compressed output

## Development

```bash
npm install
npm run dev       # Start dev server at localhost:4321
npm run build     # Build for production
npm run preview   # Preview production build
```

## Deployment

Static output in `dist/`. Deploy to Cloudflare Pages, Vercel, Netlify, or any static host.

## Branding

- Primary: Kiwi Green (`#25b381`)
- Secondary: Ocean Blue (`#3b82f6`)  
- Background: Dark (`#0a0a0f`)
- Font: Inter (body), Space Grotesk (display), JetBrains Mono (code)
